# Membaca File Teks
## Materi: Dasar-Dasar Pemrograman Modern
* Tingkat: Menengah

## Konsep
File merupakan salah satu objek yang sering digunakan dalam pemrograman. File biasanya akan memuat data. Sehingga sering disebut dengan data dalam bentuk teks.
